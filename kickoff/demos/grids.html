Standard Grid

Each .g-col element uses percentage widths, set the column size by modifying the width of the .l-container wrapper <div> in _global.scss and the $layout-max-width variables found in _variables.scss.

Offset

Move columns to the right using .g-offset* classes. Each class increases the left margin of a column by a whole column. For example, .offset2 moves a .g-col element over by two columns.
