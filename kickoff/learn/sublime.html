If you use Sublime Text, we have created some snippets to use with the framework.

How to install

If you experience problems or editor crashes please file an issue.

With Package Control:

  1. Run “Package Control: Install Package” command, find and install Kickoff Snippets plugin.
  2. Restart ST editor (if required)

Manually:

  1. Clone or download git repo into your packages folder (in ST, find Browse Packages... menu item to open this folder)
  2. Restart ST editor (if required)
